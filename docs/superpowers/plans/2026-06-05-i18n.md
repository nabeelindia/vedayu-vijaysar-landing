# i18n Multilingual Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Hindi, Tamil, and Telugu translations to Vedayu's customer-facing pages using next-i18next, with a navbar language switcher and localStorage/cookie-based persistence.

**Architecture:** `next-i18next` wraps the app; locale JSON files live in `public/locales/{en,hi,ta,te}/common.json`; `_app.js` auto-detects browser language and persists preference; a `LanguageSwitcher` component in the sticky navbar triggers `router.push` with the new locale.

**Tech Stack:** Next.js Pages Router, next-i18next, react-i18next, Google Fonts (Noto Sans Devanagari / Tamil / Telugu)

## Task 2: Config files — next-i18next.config.js + next.config.js

**Files:**
- Create: `next-i18next.config.js`
- Modify: `next.config.js`

- [ ] **Step 1: Create `next-i18next.config.js`**

```js
// next-i18next.config.js
/** @type {import('next-i18next').UserConfig} */
module.exports = {
  i18n: {
    locales: ['en', 'hi', 'ta', 'te'],
    defaultLocale: 'en',
  },
  localePath: typeof window === 'undefined' ? require('path').resolve('./public/locales') : '/locales',
  reloadOnPrerender: process.env.NODE_ENV === 'development',
};
```

- [ ] **Step 2: Update `next.config.js`**

Read the current file first, then add the i18n block. The current file has `reactStrictMode`, `images`, and `headers`. Add an i18n import and block:

```js
/** @type {import('next').NextConfig} */
const { i18n } = require('./next-i18next.config');

const nextConfig = {
  i18n,
  reactStrictMode: true,
  images: {
    domains: [],
    formats: ['image/webp', 'image/avif'],
  },
  // Security headers
  async headers() {
    return [
      {
        source: '/(.*)',
        headers: [
          { key: 'X-Content-Type-Options', value: 'nosniff' },
          { key: 'X-Frame-Options', value: 'DENY' },
          { key: 'X-XSS-Protection', value: '1; mode=block' },
          { key: 'Referrer-Policy', value: 'strict-origin-when-cross-origin' },
        ],
      },
    ];
  },
};

module.exports = nextConfig;
```

import { useState, useRef, useEffect } from 'react';
import { useRouter } from 'next/router';

const LANGUAGES = [
  { code: 'en', label: 'English',  native: 'English'  },
  { code: 'hi', label: 'Hindi',    native: 'हिन्दी'    },
  { code: 'ta', label: 'Tamil',    native: 'தமிழ்'    },
  { code: 'te', label: 'Telugu',   native: 'తెలుగు'   },
];

export default function LanguageSwitcher() {
  const router          = useRouter();
  const [open, setOpen] = useState(false);
  const ref             = useRef(null);
  const current         = LANGUAGES.find(l => l.code === router.locale) || LANGUAGES[0];

  // Close on outside click
  useEffect(() => {
    if (!open) return;
    const handler = (e) => {
      if (ref.current && !ref.current.contains(e.target)) setOpen(false);
    };
    document.addEventListener('mousedown', handler);
    return () => document.removeEventListener('mousedown', handler);
  }, [open]);

  const switchTo = (code) => {
    if (code === router.locale) { setOpen(false); return; }
    // Persist preference
    try { localStorage.setItem('vedayu_lang', code); } catch (_) {}
    // Set NEXT_LOCALE cookie (SSR picks it up on next request)
    document.cookie = `NEXT_LOCALE=${code};path=/;max-age=31536000;SameSite=Lax`;
    router.push(router.asPath, router.asPath, { locale: code, scroll: false });
    setOpen(false);
  };

  return (
    <div ref={ref} style={{ position: 'relative', display: 'inline-block' }}>
      <button
        type="button"
        onClick={() => setOpen(o => !o)}
        aria-label="Switch language"
        style={{
          display: 'flex',
          alignItems: 'center',
          gap: 4,
          background: 'none',
          border: '1.5px solid #d0c4b0',
          borderRadius: 20,
          padding: '3px 10px',
          fontSize: '.78rem',
          fontWeight: 700,
          color: '#5C3D1E',
          cursor: 'pointer',
          whiteSpace: 'nowrap',
          lineHeight: 1.4,
        }}
      >
        🌐 {current.native} ▾
      </button>

      {open && (
        <div style={{
          position: 'fixed',
          zIndex: 9999,
          background: '#fff',
          border: '1.5px solid #d0c4b0',
          borderRadius: 10,
          boxShadow: '0 4px 20px rgba(0,0,0,.13)',
          minWidth: 140,
          overflow: 'hidden',
          // Positioned dynamically via JS below — fallback to top/right
          top: ref.current
            ? ref.current.getBoundingClientRect().bottom + window.scrollY + 4
            : 50,
          left: ref.current
            ? Math.max(8, ref.current.getBoundingClientRect().left + window.scrollX - 40)
            : 'auto',
          right: ref.current
            ? 'auto'
            : 8,
        }}>
          {LANGUAGES.map(lang => (
            <button
              key={lang.code}

              onClick={() => switchTo(lang.code)}
              style={{
                display: 'flex',
